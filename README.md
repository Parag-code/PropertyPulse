# 🏠 Property Pulse - Jaipur Real Estate Price Predictor

**Property Pulse** is a data science-based web application that predicts property prices in Jaipur, India. It also provides ROI (Return on Investment) estimations and simulated environmental analysis to help users make informed real estate decisions.

---

## 🔑 Features

- **Price Prediction**: Estimate house prices based on location, area, BHK, and bathroom count.
- **ROI Estimation**: Predict potential returns over 5, 10, and 15 years.
- **Environmental Analysis**: Simulated AQI, noise level, and green area data for each location.
- **User Interface**: Built with Streamlit for a simple and interactive user experience.

---

## 🧰 Tech Stack

- Python, Pandas, NumPy
- Scikit-learn (Model Training)
- Flask (Backend API)
- Streamlit (Frontend UI)
- Pickle & JSON (Model & Metadata)

---

## 📁 Project Files

- `app.py` – Streamlit frontend
- `server.py` – Flask backend for model serving
- `util.py` – Prediction and utility functions
- `artifacts/` – Contains the trained model and column metadata
- `Jaipur_real_estate.ipynb` – Model training notebook
- `requirements.txt` – List of Python dependencies
